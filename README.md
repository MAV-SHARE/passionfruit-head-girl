# 百香果頭女孩 🥭 PASSIONFRUIT HEAD GIRL

傾斜你的手機,讓百香果滾過果肉迷宮 —— 一款 **零依賴、純前端** 的 PWA 傾斜滾球迷宮遊戲。手機優先設計,電腦也可玩。過關時,百香果頭少女會出現吃刨冰乘涼幫你慶祝!

## 🗺 系列制關卡

遊戲採**系列制**:共 11 個系列、每系列 10 關,一關一關解鎖(過關才開下一關),每關依表現獲得 1~3 星:

| 系列 | 關卡型態 | 玩法 |
|---|---|---|
| 🥭 百香果果園 | 迷宮 | 經典迷宮,收集花朵抵達漩渦 |
| 🎯 老街彈珠台 | 平衡彈珠板 | 整片坑洞無內牆,**依序通過 ①②③ 檢查點**終點才會開 |
| 🎢 滾珠珠機台 | 軌道鋼珠 | 老街經典:黃銅雙軌道蛇行來回,鋼珠被軌道夾住會鏗鏘反彈;**軌道缺口處偏了就掉出去**,掉落從檢查點續關 |
| 🍎 蘋果果園 | 迷宮 | 蘋果滾動會偏出弧線 |
| 🌉 高空獨木橋 | 平衡棧道 | 懸空蜿蜒棧道、完全無牆,**滾出邊緣就摔落** |
| 🍍 鳳梨田 | 迷宮 | 超難滾的鳳梨(靜摩擦 + 顛簸) |
| 🔔 蓮霧樹下 | 迷宮 | 春.蓮霧:又輕又彈還會飄移 |
| 🍉 西瓜田 | 迷宮 | 夏.西瓜:超重,起步慢、衝起來煞不住 |
| 🟠 柿子園 | 迷宮 | 秋.柿子:熟軟黏滯、撞牆大變形 |
| 🍓 草莓園 | 迷宮 | 冬.草莓:圓錐果形,滾動大轉彎 |
| 🌌 星雲迷宮 | 迷宮 | 高速能量球 |

迷宮關星等 = 收集數;彈珠台/獨木橋星等 = 摔落次數(0 次滿星)。

主題**視覺與物理碰撞規則都不同**:

| | 🥭 百香果(預設) | 🍎 蘋果 | 🍍 鳳梨 | 🌌 星雲 |
|---|---|---|---|---|
| 球 | 百香果(斑點隨滾動旋轉) | 紅蘋果(蒂頭葉子跟著轉) | 鳳梨(菱格果皮 + 葉冠) | 發光能量球 |
| 場景 | 紫果皮牆、金黃果肉、黑籽 | 紅果皮牆、乳白果肉、中央果核星形籽 | 褐色果皮牆、金黃果肉、同心環紋 | 霓虹牆、深空、星空 |
| 物理特性 | 幾乎不彈、阻力大、**果形不正會搖晃**、撞牆擠壓變形 | 略帶彈性,**滾動會偏出弧線不走直線** | **最難滾**:傾斜不夠推不動(靜摩擦)、幾乎不彈、一路顛簸 | 低摩擦、高彈性、速度快 |

## 📅 每日時令關卡

每天依台灣水果產季推廣一種**當令水果**(春:蓮霧鳳梨/夏:西瓜百香果/秋:柿子蘋果/冬:草莓),用該水果的模型與物理規則生成一關(由日期種子決定,全球玩家同一關),關卡型態每日輪替(迷宮/彈珠台/獨木橋)。系列選單頂部的「📅 每日時令關卡」卡片會顯示今日水果與完成狀態,搭配「每日報到」「時令達人」成就。

另外支援**自動版本更新**:偵測到新版本時會跳出「立即更新」橫幅,一鍵套用,不會被舊快取卡住。

## ✨ 特色

- **傾斜體感操作** — 使用 DeviceOrientation 感應器,手機怎麼傾斜球就怎麼滾;支援 iOS 權限請求流程
- **無限程序生成關卡** — 種子固定的迷宮演算法(遞迴回溯 + 環路打通),同一關卡號永遠是同一座迷宮,且越後面越大、越複雜
- **星星收集 & 計分** — 每關 3 顆星藏在死路深處,搭配時間獎勵與無失誤獎勵
- **黑洞陷阱** — 第 3 關起出現,掉入會被吸進去從起點重來
- **沉浸式體驗** — 自動全螢幕 + 螢幕方向鎖定、視差星空背景、粒子拖尾、碰撞震動回饋、WebAudio 即時合成音效(零音訊素材)
- **PWA 離線遊玩** — Service Worker 快取全部資源,加入主畫面後以全螢幕模式啟動
- **傾斜校準** — 一鍵把目前握持角度設為水平,躺著玩也沒問題
- **跨裝置後備操作** — 電腦可用方向鍵 / WASD / 滑鼠拖曳

## 🎮 操作

| 裝置 | 操作方式 |
|------|----------|
| 手機 / 平板 | 傾斜裝置(首次啟動會請求感應器權限) |

## 🚀 本機執行

任何靜態伺服器皆可:

```bash
python -m http.server 8123
# 或
npx serve .
```

開啟 `http://localhost:8123`。

> ⚠️ **傾斜感應器需要 HTTPS**(或 localhost)。部署到 GitHub Pages、Netlify、Vercel 等皆可直接使用。

## 📱 安裝為 App(PWA)

1. 用手機瀏覽器開啟部署後的網址(需 HTTPS)
2. Android Chrome:選單 →「加入主畫面」/「安裝應用程式」
3. iOS Safari:分享 →「加入主畫面」
4. 從主畫面開啟即為全螢幕沉浸模式

## 🛠 技術

- 純 Vanilla JS + Canvas 2D,無任何框架或建置工具
- 迷宮:mulberry32 種子隨機 + 遞迴回溯生成 + braid 環路
- 物理:圓對矩形碰撞、子步進防穿牆、反彈衰減
- 效能:迷宮牆體預渲染至離屏 Canvas,主迴圈零 shadowBlur
- 音效:Web Audio API 振盪器即時合成

## 🏆 排行榜與成就(Firebase)

遊戲內建「排行榜 / 成就」畫面(開始畫面與系列選單都有入口):

- **我的成就**:離線可用 — 暱稱(可改)、總分、總星數、通關數,以及 12 個成就徽章(初次過關、各系列全破、摘星者、全滿貫…),全部由本機進度即時計算,過關時解鎖新成就會彈出提示
- **世界排行**:需要設定 Firebase(免費方案即可),依總分排名前 50,顯示 🥇🥈🥉、暱稱、星數、分數,自己的名次會加亮;每次過關與開啟排行榜時自動上傳成績(匿名登入,玩家免註冊)

### Firebase 設定步驟

1. 到 [Firebase Console](https://console.firebase.google.com) 建立專案(不用開啟 Google Analytics)
2. 專案總覽 → 新增應用程式 → 選「網頁」→ 複製 `firebaseConfig` 物件
3. 貼到 [`js/firebase-config.js`](js/firebase-config.js)(取代 `null`)
4. Authentication → Sign-in method → 啟用 **匿名**
5. Firestore Database → 建立資料庫(正式版模式)→ 規則貼上:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /players/{uid} {
      allow read: if true;
      allow write: if request.auth != null && request.auth.uid == uid
                   && request.resource.data.name is string
                   && request.resource.data.name.size() <= 20
                   && request.resource.data.totalScore is number;
    }
  }
}
```

6. 部署(`firebase-config.js` 走網路優先,改完即生效,不必等版本更新)

> 未設定 Firebase 時,排行榜自動降級為離線模式,遊戲其他功能完全不受影響;Firebase SDK 採動態載入,離線遊玩不會嘗試下載。
